# 函数作用域和块作用域

3.1 函数中的作用域

函数作用域的含义是指，属于这个函数的全部变量都可以在整个函数的范围内使用及复用(事实上在嵌套的作用域中也可以使用)。

3.2 隐藏内部实现

从所写的代码中挑选出一个任意的片段，然后用函数声明的方式对他进行包装，实际上就是把这些代码“隐藏”起来了。

为什么隐藏？

它们大都是从最小特权原则中引申出来的，也叫最小授权或最小暴露原则。这个原则是指在软件设计中，应该最小限度地暴露必要内容，而将其他内容都“隐藏”起来，比如某个模块或对象的API 设计。
